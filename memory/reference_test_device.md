Test device: Samsung Galaxy SM S721B, Android 16, ARM64, USB-attached with debugging enabled.
Flutter device id: `R5CY604D5BV`.

Run on device: `flutter run -d R5CY604D5BV` (debug APK with hot reload).
List devices: `flutter devices`.

This is the primary phone for verifying mobile-first behavior. Desktop (Windows) builds are secondary — they exist only because Flutter gives them for free; user does not optimize for desktop.
